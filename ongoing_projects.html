Solar Charging Array
Stage: Design

Synopsis:
The purpose of this project is to design and fabricate a charging array for power tools. A minimum of two arrays will be disigned, and at least one will run off of a solar panel.
Purpose:
  • Gain more experience with solar and power systems.
  • Charge power tools in an efficient manner.
Design Process:
  • Currently, the project is in the disign stage.
  • The frame of the charger will likely be 3d printed, and the ports will be ordered online.
  • Research is currently being conducted on the best option for the solar panel and power systems.

Physics Simulation Program
Stage: Design

Synopsis:
Create a program that can run different physics simulations. The first part of this project will be to simulate electron clouds around atomic nuclei.
Purpose:
  • Gain more experience coding large projects.
  • Demonstrate coding abilities.
  • Create a tool to use to demonstrate physics phenomena visually
Design Process:
  • Currently, the project is in the disign stage.
  • All the calculations will be done in C++. Visuals will be handled by either python or C++.
  • There will be a full GUI that will allow users to select which simulation they would like to run. For the electron cloud simulation, there will be a GUI that allows users to select different energy levels and cross-sections.
